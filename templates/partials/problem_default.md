# Background
Special for beginners, ^_^

# Description
Given two integers x and y, print the sum.

# Format

## Input
Two integers x and y, satisfying 0 <= x, y <= 32767.

## Output
One integer, the sum of x and y.

# Sample 1

## Input
```
123 500
```

## Output
```
623
```
